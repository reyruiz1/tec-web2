TEC WEB

Ingenieria de sistemas

HTML

Es un lenguaje de marcado esencial para la creación de páginas web. Define la estructura y contenido de una página mediante etiquetas que indican cómo mostrar elementos como texto, imágenes y enlaces.

CSS

Es un lenguaje de diseño utilizado en desarrollo web para dar estilo y formato a las páginas, controlando la apariencia de elementos como colores, fuentes, márgenes y disposición.

JS

Es un lenguaje de programación utilizado principalmente en el desarrollo web para agregar interactividad y dinamismo a las páginas. Permite crear efectos visuales, validar formularios, realizar acciones en respuesta a eventos y más, todo en el navegador del usuario.
